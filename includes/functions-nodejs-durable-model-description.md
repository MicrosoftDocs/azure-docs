> [!NOTE]
> Version 4 of the Node.js programming model for Azure Functions is generally available. The v4 model is designed to provide a more flexible and intuitive experience for JavaScript and TypeScript developers. For more information about the differences between v3 and v4, see the [migration guide](../articles/azure-functions/functions-node-upgrade-v4.md).
