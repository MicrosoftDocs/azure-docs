#### VPN type - Resource Manager deployment model

|      | **Route-based**    | **Policy-based** |
|-----------------------------------|--------------------|------------------|
| **Site-to-Site**                  | yes                | yes              |
| **VNet-to-VNet**                  | yes                | no               |
| **Multi-Site**                    | yes                | no               |
| **S2S and ExpressRoute coexist**  | not yet available  | no               |
| **Point-to-Site**                 | yes                | no               |
| **Classic to Resource Manager**   | yes                | no               |
