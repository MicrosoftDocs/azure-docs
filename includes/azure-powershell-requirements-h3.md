### Requirements

* If you choose to use PowerShell locally:
  * [Install the Az PowerShell module](/powershell/azure/install-az-ps).
  * Connect to your Azure account using the
    [Connect-AzAccount](/powershell/module/az.accounts/connect-azaccount) cmdlet.
* If you choose to use Cloud Shell:
  * See [Overview of Azure Cloud Shell](https://docs.microsoft.com/azure/cloud-shell/overview) for
    more information.
