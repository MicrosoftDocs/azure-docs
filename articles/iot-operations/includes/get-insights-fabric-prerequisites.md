You also need the following Fabric resources:

- A Microsoft Fabric subscription. In your subscription, you need access to a workspace with **Contributor** or higher permissions.

For more information, see [Enable tenant settings in the admin portal](/fabric/real-time-intelligence/dashboard-real-time-create#enable-tenant-settings-in-the-admin-portal).
