When using Text Analytics, the documents in your request must not exceed the following character limits.

Synchronous calls and Text Analytics for health calls: 5,120 characters in one document.
Asynchronous calls: 125,000 characters per document.

See the [data limits article](../concepts/data-limits.md) for more information.
