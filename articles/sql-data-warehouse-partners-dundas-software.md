#Dundas Software
![][logo]
##Who we are
[Dundas Data Visualization][website] is a leading, global provider of Business Intelligence and Data Visualization software providing Dashboards, Reporting and Visual Data Analytics, with seamless integration into business applications.  Our mission is to provide organizations with the most flexible, innovative and scalable Business Intelligence and Data Visualization solutions, through a fully embeddable BI platform, enabling user access to all their data for better decisions and faster insights.     

##What We Offer
Dundas BI™ - Dashboards, Reporting & Visual Data Analytics all with Seamless Integration.

Dundas BI is an innovative Business Intelligence (BI) and Data Visualization software with customizable dashboards, reporting and visual data analytics. Dundas BI is the most flexible BI platform, with seamless Dundas BI integration into your business applications, allowing you to deliver real-time visual access and analytics for all your data, information and results, for better decisions and faster insights. 

Dundas BI provides a consistent user experience, views and options across any data source and on all devices.  With tailored, self-service by user type, you can easily and quickly perform data discovery, on-the-fly data analysis and create professional dashboards and reports. The Dundas BI platform was designed with extensibility in mind, supporting many levels of customizations, and can be seamlessly integrated and embedded into your existing systems, applications and administration.

Dundas BI provides an enterprise-ready BI platform with ease of management, full control and the security and governance needed for large-scale integration and deployment. 

##Dundas BI for Azure SQL Data Warehouse
Dundas BI is built on top of Microsoft technologies – we’re Microsoft friendly.  

Dundas has always offered seamless, out-of-the-box integration with existing Microsoft technologies like SQL Server, SQL Server Analysis Services, Azure SQL Database, Microsoft SharePoint and Excel. With the announcement of the new Azure SQL Data Warehouse, Dundas BI offers the same seamless integration and full support, enabling users to directly connect to and quickly access all of their data, whether Dundas BI is installed on the Azure Cloud or on premise. This out-of-the box integration between Dundas BI and Azure SQL Warehouse makes it easier to develop and faster to deploy. The built-in connectivity with ease of maintenance and administration in Dundas BI allows organizations to quickly realize the business value of their data on the Azure cloud.
