In this article:

- [View, update, or delete NFS volumes](#view_update_delete_volumes)
- [View, update, or delete mount targets](#view_update_delete_mount_targets)
- [Edit volume attributes](#edit_volume_attributes)

## <a name="view_update_delete_volumes"></a>View, update, or delete volumes  
Use the Browse NFS Volume view to view, update, or delete existing volumes.  

You must delete all sub-objects of a volume (for example, mount targets) before you can delete the volume itself.
