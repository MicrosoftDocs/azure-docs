# Collecting data from Microsoft Cloud App Security 

You can stream logs from [Cloud App Security](https://docs.microsoft.com/cloud-app-security/what-is-cloud-app-security) into Security Insights with a single click.

## Prerequisites

- User with global administrator permissions

## Connect to Cloud App Security

If you already have Cloud App Security, make sure it is [enabled on your network](https://docs.microsoft.com/cloud-app-security/getting-started-with-cloud-app-security).
If Cloud App Security is deployed and ingesting your data, the alert data can easily be streamed into Azure Security Insights.


1. In Azure Security Insights, select **Data collection** and then click the **Cloud App Security** tile.

2. Click **Connect**.
