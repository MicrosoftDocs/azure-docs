#### [bash](#tab/terminal-bash)

```azurecli
az webapp deploy \
    --name $APP_SERVICE_NAME \
    --resource-group $RESOURCE_GROUP_NAME  \
    --src-path <zip-file-path> \
    --type zip
```

#### [PowerShell terminal](#tab/terminal-powershell)

```azurecli
az webapp deploy `
    --name $APP_SERVICE_NAME `
    --resource-group $RESOURCE_GROUP_NAME  `
    --src-path <zip-file-path> `
    --type zip
```
