# Microsoft Defender Threat Intelligence connector for Microsoft Sentinel

Microsoft Sentinel provides you with the capability to import threat intelligence generated by Microsoft to enable monitoring, alerting, and hunting. Use this data connector to import Indicators of Compromise (IOCs) from Microsoft Defender Threat Intelligence (MDTI) into Microsoft Sentinel. Threat indicators can include IP addresses, domains, URLs, and file hashes, etc.
