# Customer intent: "As an IT administrator who manages virtual machines, I want to configure patch orchestration schedules so that I can ensure my systems are updated and maintain business continuity."
---


## Prerequisites

#### [Azure VMs](#tab/avms)

**Patch Orchestration** must be set to **Customer Managed Schedules**.

#### [Azure Arc-enabled VMs](#tab/arcvms)

There are *no prerequisites* for patch orchestration. However, you must associate a schedule with the VM for scheduled patching.
