# Install Visual Studio 2015 and SSDT for SQL Data Warehouse

To develop queries for SQL Data Warehouse, we recommend using Visual Studio 2013 or later in combination with SQL Server Data Tools (SSDT). These steps show you how to install both Visual Studio and SSDT. When you are finished, you will be ready to connect and query your SQL Data Warehouse database.
 
## Download and Install Visual Studio

You can use either Visual Studio 2013 or Visual Studio 2015 with SQL Data Warehouse. If you already have Visual Studio 2013 installed, just skip to the next step. 

To install Visual Studio 2015:

1. [Download Visual Studio 2015](https://www.visualstudio.com/downloads) from Visual Studio online. 
2. Install by following the [Installing Visual Studio](https://msdn.microsoft.com/library/e2h7fzkw.aspx) guide on MSDN and choose the default configurations.

## Download and Install SQL Server Data Tools (SSDT) Preview

Regardless of the version of Visual Studio that you have installed, you need the latest preview version of SQL Server Data Tools (SSDT) in order to interact with your SQL Data Warehouse database.

To install SQL Server Data Tools, latest preview version:

1. [Download SQL Server Data Tools Preview](https://msdn.microsoft.com/mt429383.aspx) for either Visual Studio 2013 or 2015.
2. Install by following the installation instructions on the download site.

## Next steps

Now that you have Visual Studio and the latest preview version of SSDT, you are ready to [connect and query](./sql-data-warehouse-get-started-connect-query.md).
