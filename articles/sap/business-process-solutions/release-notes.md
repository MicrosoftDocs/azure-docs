# Release notes for Business Process Solutions

The following document outlines the changes to Business Process Solutions. Changes can include bug fixes and updates to existing notebooks, pipelines, and environments. The document is updated with each release of the solution and is intended to keep users informed about the latest updates and breaking changes to the solution.

## Version 1.0.1

- Added note to all the notebooks. This note indicates that the notebooks shouldn't be modified. If users want to modify the notebooks, they should create a copy of the notebook and modify the copy instead.

## Version 1.0.2

- Added new helper functions to common environment file.
- Updated bps_ecc_mirror_db_view_creation.ipynb notebook, fixed issues with existing views, and added new views for text tables.
- Added new notebooks bps_nb_ecc_hierarchy_processing.ipynb and bps_nb_ecc_process_fiscal_periods.ipynb for processing hierarchy and fiscal periods data from ECC system.
- Added minor bug fixes and improvements to other existing notebooks.
