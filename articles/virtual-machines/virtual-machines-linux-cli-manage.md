# Equivalent Azure Resource Manager (arm) mode and Azure Service Management (asm) mode commands for VM tasks with the Azure CLI

This article shows equivalent Azure Command-Line Interface (Azure CLI) commands to create and manage Linux VMs in Azure Service Management (asm) mode and Azure Resource Manager (arm) mode. Use this as a handy guide to migrate scripts from one command mode to the other. This article is also available for [Windows VMs](virtual-machines-windows-cli-manage.md).
