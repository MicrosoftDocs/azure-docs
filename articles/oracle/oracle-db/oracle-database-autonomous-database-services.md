# Oracle Autonomous AI Database in Oracle AI Database@Azure

Oracle AI Database@Azure provides you with a seamless integration of Oracle Autonomous AI Database resources within your Azure cloud environment.

You access your instance of Oracle AI Database@Azure through the Azure portal. Then, you create and manage Oracle Autonomous AI Database resources in the Azure portal. For more Oracle Autonomous AI Database functionality, you have direct access to the Oracle Cloud Infrastructure (OCI) portal.

Oracle AI Database@Azure has different IP address requirements than Oracle Cloud Infrastructure (OCI). Oracle AI Database@Azure reserves 13 IP addresses for the client subnet versus 3 for OCI requirements.

For more information about Oracle Autonomous AI Database beyond implementation and use within Oracle AI Database@Azure, see [Using Oracle Autonomous AI Database Serverless](https://docs.oracle.com/en/cloud/paas/autonomous-database/serverless/adbsb/index.html).
