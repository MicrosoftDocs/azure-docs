# What is the Azure Dev/Test offer?

This article explains the Azure Dev/Test benefit that comes with your Visual Studio Subscription benefits. 

With this benefit, you get monthly Azure credits to run development and test workloads at no cost within the credit limit. Credit amounts vary by subscription level, and usage stops when the credit is exhausted if no payment method is added.

Azure Dev/Test also provides discounted pricing for Azure services - such as virtual machines, Azure SQL Database, and App Service - to support non-production workloads. Discounted pricing is available with active Visual Studio Subscriptions.

## Who can benefit? 

With Azure Dev/Test, these Visual Studio subscribers can use Azure:
+ [Individuals can use Azure as personal sandbox for dev/test](quickstart-individual-credit.md). 
+ [Enterprises can use Azure for team development and collaboration](quickstart-create-enterprise-devtest-subscriptions.md).
