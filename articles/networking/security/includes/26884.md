Azure Front Door Web Application Firewall (WAF) provides bot protection through the Bot Manager rule set, available exclusively in the Premium SKU, which identifies and categorizes automated traffic at the global edge network. Without bot protection, threat actors can deploy automated attacks including credential stuffing, web scraping, inventory hoarding, and application-layer distributed denial of service (DDoS) attacks. The Bot Manager rule set categorizes bots into known good bots, known bad bots, and unknown bots, allowing security teams to configure appropriate actions for each category. Bad bots can be blocked or challenged with CAPTCHA, while legitimate bots like search engine crawlers are allowed through. Without bot protection, organizations lack visibility into bot traffic patterns and cannot distinguish between human users and automated clients.
