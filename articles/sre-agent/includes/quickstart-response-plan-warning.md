> [!IMPORTANT]
> When you first connect an incident platform, a default **quickstart** response plan is automatically created. If you create your own response plans, the quickstart plan runs alongside them and can cause incidents to be routed to the wrong custom agent or processed twice. To avoid conflicts, go to **Builder** > **Incident response plans**, switch to **Table view**, and delete the quickstart plan.
